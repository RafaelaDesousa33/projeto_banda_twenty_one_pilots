Projeto Banda – Twenty One Pilots

Projeto desenvolvido como atividade acadêmica com o objetivo de criar uma página web estática sobre uma banda musical, utilizando HTML e CSS.

A banda escolhida foi Twenty One Pilots, devido à sua identidade visual marcante e relevância no cenário musical.

Objetivo do Projeto

Praticar a estruturação de páginas com HTML semântico

Aplicar estilização utilizando CSS

Trabalhar organização de layout

Utilizar boas práticas de separação entre estrutura e estilo

Desenvolver noções de hierarquia visual

Tecnologias Utilizadas

HTML5

CSS3

Visual Studio Code

Live Server

Estrutura do Projeto

projeto_banda/
│
├── index.html
├── style.css
└── images/
  └── imagem da banda

Conceitos Aplicados

Estrutura HTML semântica

Uso de div, figure e figcaption

Hierarquia de títulos (h1, h2)

Organização com classes

Box Model

Containers para centralização de conteúdo

Separação entre HTML e CSS

Funcionalidades da Página

Título principal da banda

Descrição introdutória

Imagem em destaque dos integrantes

Seção de biografia

Estilização com cores inspiradas na identidade visual da banda

Como Executar o Projeto

Baixe ou clone o repositório

Abra a pasta no Visual Studio Code

Execute o arquivo index.html

Ou utilize a extensão Live Server para rodar localmente

Aprendizados

Durante o desenvolvimento, foram reforçados conceitos fundamentais do front-end, como organização de código, estruturação limpa, criação de layout simples e aplicação prática do Box Model.
